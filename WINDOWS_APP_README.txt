GB1 Multi-Mutant App

How to launch:
1. Extract this folder from the zip file.
2. Double-click "GB1 Multi-Mutant App.bat".
3. A browser window opens automatically.
4. Keep the black launcher window open while using the app.
5. Close the launcher window when you are done.

What the app does:
- Loads the packaged GB1 binding, stability, double-mutant, and ESM/LM score CSV files.
- Runs the no-LDA co-optimization workflow.
- Shows Inputs / Run, Outputs / Results, and a 3D Mutation Module.
- Lets you inspect selected red or blue graph points and map the mutation sites onto a simplified GB1 fold.

Important:
The 3D Mutation Module is an interpretive structure map. It highlights mutation positions on a simplified GB1 fold and compares WT vs mutated sequence. It does not run molecular dynamics, docking, or a new folding prediction.
